Материалы по курсу "Параллельные вычисления для научных расчетов"

Курс длится 1 семестр, в конце семестра зачет/экзамен. В неделю 1 лекция.
Курс поддержан грантом Благотвортительного фонда В. Потанина

Цель преподавания дисциплины

Изучение принципов и методов параллельного программирования, применение изученных методов для решения вычислительно затратных научных задач.

Задачи изучения дисциплины

  1. Освоение стандартов OpenMP и MPI в объёме, достаточном для решения практических задач.
  2. Знакомство с параллельными алгоритмами на больших графах (выделение сообществ, поиски путей и т. п.). Параллельное программирование для задач с интенсивным доступом к данным.
  3. Знакомство с параллельными методами линейной алгебры, параллельными методами решения обыкновенных дифференциальных уравнений и уравнений в частных производных. Параллельное программирование для вычислительно-интенсивных задач

Место дисциплины в системе высшего профессионального образования

В курсе "Параллельные вычисления для научных расчетов" используются знания, полученные студентами при изучении курсов (в скобках указана трудоемкость изученных курсов)
  1. Операционные системы (1 семестр, лекции + практики),
  2. Языки и технологии программирования (2 семестра, лекции + практики),
  3. Аппаратные средства вычислительной техники (1 семестр, лекции + практики)
  4. Архитектура ЭВМ (1 семестр, лекции),
  5. Методы приближенных вычислений (2 семестра, лекции + практики),
  6. Дискретная математика (2 семестра, лекции + практики),
  7. Комбинаторные алгоритмы (2 семестра, лекции + практики),
  8. Параллельные вычисления (1 семестр, лекции + практики).
Одновременно с курсом "Параллельные вычисления для научных расчетов" студентам читаются курсы:
  1. Программирование на GPU,
  2. Программирование для сопроцессоров Intel Xeon Phi
Знания, полученные при изучении курса "Параллельные вычисления для научных расчетов", используются при изучении курсов:
  1. Оптимизация производительности вычислительных приложений,
  2. Моделирование живых систем.

Требования к уровню освоения содержания курса

После успешного прохождения учебного курса студенты должны:
  1. оценивать пригодность вычислительной системы для решения требуемой прикладной задачи;
  2. знать, применять и адаптировать классические методы параллельного решения практически важных задач;
  3. учитывать специфику задачи (например, зависимость от объема данных) при разработке алгоритмов, обладающих высокой степенью параллелизма;
  4. уметь проводить декомпозицию параллельного алгоритма на архитектуру вычислительной системы;
  5. программно реализовывать разработанный метод, проводить его тестирование и профилировку;
  6. уметь использовать специализированные библиотеки, типа MKL, для научных расчетов.

Используемое оборудование

Для нужд курса будет использоваться вычислительный кластер Уран Института математики и механики УрО РАН. Так же будет использоваться вычислительный кластер УрФУ, состоящий из 12 вычислительных узлов на базе CPU Intel Xeon, а также сопроцессоров Intel Xeon Phi 5110P и ускорителей вычислений Nvidia Tesla K20X.

Программа курса

возможны изменения
  1. Введение в высокопроизводительные вычисления;
  2. Параллелизм потоков и OpenMP;
  3. Параллелизм процессов и MPI;
  4. Структурное параллельное программирование;
  5. Умножение и декомпозиция больших матриц: разреженных и плотных;
  6. Параллельные прямые и итерационные методы решение систем линейных алгебраических уравнений;
  7. Параллельные численные методы решения систем обыкновенных дифференциальных уравнений;
  8. Параллельные конечно-разностные схемы решения дифференциальных уравнений в частных производных;
  9. Экономичные схемы решения задач математической физики;
  10. Распараллеливание методов типа Шварца для задач математической физики на "сложных" областях;
  11. Data intensive задачи. Высокопроизводительная обработка графов;
  12. Параллельный поиск в ширину на графе;
  13. Параллельный поиск кратчайших путей на графе;
  14. Параллельный поиск сообществ в графах;
  15. Параллельное определение метрик центральности в графах.

Формальные требования для получения зачета

В течение семестра будут предложены задачи. Для получения зачета/экзамена необходимо сдать задачи до конца сессии. Студентам рекомендовано объединяться в группы по 3-4 человека и выполнять задачи коллективно. Сданные задачи и полученные баллы можно отслеживать в google-доках.

Консультации

Консультации проводятся на кафедре вычислительной математики по расписанию. На консультации можно не только задать вопросы, но и сдать задачи.
